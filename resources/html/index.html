Core features

Standalone

No external styles, fonts, icons or scripts — everything in this single HTML file.

Responsive

Grid-based layout adapts from wide desktop down to small phones seamlessly.

Accessible

Semantic structure and sensible focus order for keyboard users and screen readers.

Customizable

Change CSS variables at the top to quickly rebrand colors, radii and typography.

Light/Dark

Built-in theme toggle (uses a tiny bit of JS — can be removed if you prefer pure-CSS).

Small footprint

The whole page is loaded from a single file — ideal for demos and static hosting.
